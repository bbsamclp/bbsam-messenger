# Screenshot-Vorlagen

## Verwendung

1. Öffne die SVG-Vorlage in einem Grafikprogramm (Inkscape, Figma, etc.)
2. Füge den eigentlichen App-Screenshot in den markierten Bereich ein
3. Passe die Überschrift/Beschreibung an den jeweiligen Screenshot an
4. Exportiere als PNG in der benötigten Auflösung

## Vorlagen

| Datei | Format | Verwendung |
|-------|--------|------------|
| `template-phone.svg` | 1080x1920 (9:16) | Google Play, App Store (Smartphone) |
| `template-tablet.svg` | 2048x2732 (3:4) | App Store (iPad) |
